# 📦 E-Commerce Delivery Prediction App

## 🚀 Project Overview
This project predicts whether an e-commerce product will be delivered on time or not using Machine Learning.

## 🧠 Model Used
- Gradient Boosting Classifier

## ⚙️ Features Used
- Warehouse Block
- Mode of Shipment
- Customer Care Calls
- Customer Rating
- Cost of Product
- Prior Purchases
- Product Importance
- Gender
- Discount Offered
- Weight in grams

## 🔧 Tech Stack
- Python
- Pandas
- Scikit-learn
- Streamlit

Users can input product and customer details to predict delivery status.
